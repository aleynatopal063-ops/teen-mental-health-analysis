# Teen Mental Health Analysis

This project studies how social media affects teenagers' mental health.

## 📊 Dataset
The dataset includes:
- Social media usage (hours)
- Sleep hours
- Stress level
- Anxiety level
- Physical activity

## 📈 Analysis
We used:
- Data visualization (matplotlib, seaborn)
- Correlation analysis
- Scatter plots

## 🤖 Machine Learning
We used a Random Forest model to predict depression risk.

Accuracy: ~99%

## 📌 Conclusion
Social media usage is related to mental health factors like stress, anxiety, and sleep.

This project shows how daily habits can affect mental health.
